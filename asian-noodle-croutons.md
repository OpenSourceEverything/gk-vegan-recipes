 // Ingredients:
// - 1 cup whole wheat flour
// - 1/2 tsp active dry yeast (fast rise)
// - 1/4 tsp salt
// - 1/4 cup warm water (110°F)
// - 1/2 tsp sesame oil (optional, for flavor)

// Instructions:
// 1. Dissolve yeast in warm water. 
//    Let sit 5 minutes until frothy.
// 2. Mix flour and salt in a bowl. Add yeast mixture and sesame oil.
// 3. Knead into a smooth dough (5-7 min). Rest 20-30 min.
// 4. Roll dough thin (1-2 mm). Cut into thin noodles.
// 5. Lay noodles on a baking sheet lined with parchment paper.
// 6. Lightly brush noodles with oil or spray with cooking spray.
// 7. Preheat oven to 375°F (190°C).
// 8. Bake noodles for 8-12 min, flipping halfway, until golden and crispy.
// 9. Cool completely. Store in an airtight container.

// Notes:
// - Adjust bake time for desired crispiness.
// - Brushing with sesame oil enhances flavor.
